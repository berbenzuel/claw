# claw
Rust  web assembly 



# Dependencies

install cargo dependencies:
>     cargo install cargo-watch
>     cargo install wasm-pack

install web dependecies:
- nodejs npm: [Nodejs npm](https://nodejs.org/en/download)
>     nmp install vite


# How to start the whole thing

To easily run the server just execute cargo-run-dev.sh in frontend/src-rust
and also start Vite server using classic "npm run dev" in frontend/frontend folder
